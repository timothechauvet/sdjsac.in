# Static Assets Directory

This folder is served statically at `/images/` (e.g., `http://localhost:3000/images/example.webp`).

Place your actual images here to replace the placeholder boxes on the site:

1. **UPI QR Code**: Save as `qr-code.webp` (or `.png`/`.jpg`) and update the `src` attribute in `app/page.js` to `/images/qr-code.webp`.
2. **Statue / Ceremony Image**: Save as `statue.webp` and update the image container `src` attribute in `app/page.js`.
3. **Gallery Images**: Save your gallery images here (e.g., `gallery-1.webp`, `gallery-2.webp`, etc.) and reference them in `app/page.js`.
